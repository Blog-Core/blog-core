# blog-core

A CMS/blog framework for [Swi-Prolog](http://swi-prolog.org/).

## Features

 * Multiple users.
 * Multilanguage support.
 * Markdown for content.
 * Threaded comments.
 * Built-in view caching.
 * Custom URLs.
 * Administration API.
 * Administration UI.
 * Data migrations.

## Getting started

Visit <http://blog-core.net/> for the further info.
